# addDigits
Реализуйте и экспортируйте по умолчанию функцию, которая работает следующим образом:  
Дано неотрицательное целое число num. Складывать все входящие в него цифры до тех пор, пока не останется одна цифра.
```
addDigits(10); // 1
addDigits(19); // 1
addDigits(38); // 2
addDigits(1259); // 8
```
